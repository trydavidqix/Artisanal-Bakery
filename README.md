## 📝 Descrição
Site institucional para a padaria artesanal "Pão com Amor", desenvolvido com HTML, CSS e JavaScript puro. O projeto apresenta um design moderno e responsivo, focado na experiência do usuário e na apresentação dos produtos artesanais.

## ✨ Funcionalidades

- 🎥 Vídeo de fundo na seção hero
- 📱 Menu mobile responsivo
- 🖼️ Galeria de produtos com slider
- 👥 Seção de equipe com cards interativos
- 📨 Formulário de contato funcional
- ⚡ Animações suaves de scroll
- 🔄 Loading screen personalizado
- 🎨 Design totalmente responsivo

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome (ícones)
- Google Fonts (Poppins)

## 📱 Responsividade

O site é totalmente responsivo e se adapta aos seguintes breakpoints:

- 📱 Mobile: < 768px
- 📱 Tablet: 768px - 1024px
- 💻 Desktop: > 1024px

## ✨ Seções do Site

1. **Home**
   - Vídeo de fundo
   - Chamada principal
   - Botão CTA

2. **Sobre Nós**
   - História da padaria
   - Diferenciais
   - Valores

3. **Produtos**
   - Grid de produtos
   - Preços
   - Descrições

4. **Galeria**
   - Slider interativo
   - Navegação por dots
   - Touch support

5. **Nossa Equipe**
   - Cards dos membros
   - Links para redes sociais
   - Hover effects

6. **Contato**
   - Formulário de contato
   - Informações de contato
   - Localização
